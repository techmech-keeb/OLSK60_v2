# OLSK60 マウス機能操作マニュアル

このドキュメントは、OLSK60 v2 のトラックポイント速度設定に関する詳細マニュアルです。トラックポイントの速度を自分好みに微調整したい方を対象としています。  
基本操作や他の機能については[キーボード操作ガイド](OLSK60_user_guide.md)をご参照ください。

---

## クイックリファレンス

すべての操作はレイヤー2（MO(2)キーを押しながら）で行います。

| キー | 機能 |
|------|------|
| 1～3 | 固定プロファイル切替（Precision / Balanced / Fast） |
| 4～5 | カスタムバンク切替（Custom Precision / Custom Fast） |
| Q / W | 基本速度を上げる / 下げる（カスタムバンク選択時のみ） |
| A / S | 加速度を上げる / 下げる（カスタムバンク選択時のみ） |
| D / F | 減速度を上げる / 下げる（カスタムバンク選択時のみ） |
| Y / U / I | オートレイヤー遅延（短 / 中 / 長） |
| O | オートレイヤー ON/OFF |

## トラックポイントの基本操作

### カーソル移動

トラックポイントを指先で軽く押すことでカーソルを移動できます。押す力の強さに応じてカーソルの移動速度が変化します。

### マウスボタン

最下段に配置された専用キーで操作します。マウス左ボタン・マウス右ボタンは**レイヤーに関係なく常時有効**です。

| 操作 | キー |
|------|------|
| 左クリック | マウス左ボタン（最下段） |
| 右クリック | マウス右ボタン（最下段） |

### スクロールモード

レイヤー3（マウスレイヤー）がアクティブな状態で、スクロールモードキーを押しながらトラックポイントを動かすと、カーソル移動ではなくスクロール操作になります。

- 上下方向：ページの縦スクロール
- 左右方向：ページの横スクロール

### Depress機能（押し下げクリック）

トラックポイントを軽く押し下げることでクリック操作が可能です。起動時に自動的に有効化されます。

## 速度プロファイル

OLSK60 は 5 つの速度プロファイルを備えています。3 つの固定プロファイルと、ユーザーが自由に調整できる 2 つのカスタムバンクです。

### プロファイル一覧

| # | プロファイル名 | 速度の目安 | 編集 |
|---|--------------|-----------|------|
| 1 | Precision（精密） | 低速（1x） — 細かい操作向け | 不可 |
| 2 | Balanced（標準） | 中速（2x） — 汎用 | 不可 |
| 3 | Fast（高速）**※デフォルト** | 高速（約3.3x） — 広い画面向け | 不可 |
| 4 | Custom Precision（カスタム精密） | 初期値は Precision と同じ | 可能 |
| 5 | Custom Fast（カスタム高速） | 初期値は Fast と同じ | 可能 |

速度の倍率はPrecisionを基準（1x）とした目安です。カスタムバンクは調整後の値がEEPROMに保存されます。

### プロファイルの切り替え

レイヤー2（MO(2)キーを押しながら）で、数字キー1～5を押して切り替えます。

![OLSK60 レイヤー2 設定キーの物理配置図](images/mouse_setting.png)

| キー | プロファイル | LED色 |
|------|-------------|-------|
| 1 | Precision | 青 |
| 2 | Balanced | 緑 |
| 3 | Fast | 赤 |
| 4 | Custom Precision | 紫 |
| 5 | Custom Fast | オレンジ |

切り替え時、対応するプロファイル色でLEDが2回点滅します。

### 3つのパラメータについて

| パラメータ | 効果 |
|-----------|------|
| 基本速度 | カーソル移動の全体的な速さ。大きいほど速い |
| 加速度 | トラックポイントを強く押した時の速度の増加率 |
| 減速度 | トラックポイントから力を抜いた時の速度の減衰率 |

## カスタムバンクの調整

Custom Precision（4）と Custom Fast（5）は、ユーザーが3つのパラメータを個別に調整できるプロファイルです。

> ⚠️ **重要:** カスタムバンク（4または5）を選択中のみ調整可能です。固定プロファイル（1～3）選択中に調整しようとすると、LEDが赤く3回点滅し、操作が拒否されます。

### 調整手順

以下の手順で、レイヤー2を使ってカスタムバンクを調整します。

1. レイヤー2で「4」または「5」を押し、カスタムバンクを選択します（対応色でLEDが2回点滅）。
2. 以下のキーでパラメータを調整します。

   | キー | 機能 | 成功時LED |
   |------|------|----------|
   | Q | 基本速度を上げる（1ステップ） | 明るい緑が1回点滅 |
   | W | 基本速度を下げる（1ステップ） | 暗い緑が1回点滅 |
   | A | 加速度を上げる（1ステップ） | 明るい緑が1回点滅 |
   | S | 加速度を下げる（1ステップ） | 暗い緑が1回点滅 |
   | D | 減速度を上げる（1ステップ） | 明るい緑が1回点滅 |
   | F | 減速度を下げる（1ステップ） | 暗い緑が1回点滅 |

3. 設定は自動保存されるので、そのまま使い始められます。

### 調整の範囲

各パラメータには上限と下限があり、限界値に達するとそれ以上の調整は反映されません。

| パラメータ | 調整可能なステップ数（目安） |
|-----------|------------------------|
| 基本速度 | 約30ステップ |
| 加速度 | 約15ステップ |
| 減速度 | 約30ステップ |

ステップ数はカスタムバンクの初期値からの調整余地を示しています。初期値から上下どちらの方向にも調整できます。

### 使い方の例

2つのカスタムバンクにそれぞれ異なるチューニングを保存し、作業に応じて切り替えて使えます。

- **Custom Precision（4）** → 画像編集やCADなど精密な操作用に、基本速度を低くし減速度を高めに設定
- **Custom Fast（5）** → マルチディスプレイ環境用に、基本速度と加速度を高めに設定

## オートレイヤー

トラックポイントを操作すると、自動的にレイヤー3（マウスレイヤー）に切り替わる機能です。詳しい動作の仕組みは[キーボード操作ガイドのオートレイヤーセクション](OLSK60_user_guide.md#オートレイヤー機能)をご参照ください。

### 解除タイミングの調整

レイヤー2で、トラックポイント操作停止後にレイヤー3が解除されるまでの時間を3段階から選べます。

| キー | 設定 | 遅延時間 | LED色 | 用途 |
|------|------|---------|-------|------|
| Y | 短い | 150ms | 赤 | キー入力を優先したい場合 |
| U | 中間 | 400ms | 緑 | バランスの取れた設定 |
| I | 長い | 800ms | 青 | マウス操作を優先したい場合 **※デフォルト** |

### オートレイヤーの ON/OFF

レイヤー2 + O キーで機能自体を切り替えられます。

| 状態 | LED表示 |
|------|--------|
| ON | 緑が2回点滅 |
| OFF | 赤が2回点滅 |

OFFにしても、トラックポイントのカーソル移動やマウスボタンは通常どおり動作します。

## LEDリファレンス

### 速度プロファイルの色

レイヤー3がアクティブな時、選択中のプロファイルに応じた色で表示されます。

| プロファイル | LED色 |
|-------------|-------|
| Precision | 青 |
| Balanced | 緑 |
| Fast | 赤 |
| Custom Precision | 紫 |
| Custom Fast | オレンジ |

### 設定操作時のフィードバック

| 操作 | LED表示 |
|------|--------|
| プロファイル変更 | 対応色が2回点滅 |
| カスタムパラメータ増加 | 明るい緑が1回点滅 |
| カスタムパラメータ減少 | 暗い緑が1回点滅 |
| 操作拒否 | 赤が3回点滅 |
| オートレイヤー遅延変更 | 設定に対応した色が1回点滅 |

## 設定の保存

- 速度プロファイルの選択、カスタムバンクのパラメータ、オートレイヤーの設定は自動的にEEPROMに保存されます。
- キーボードの電源を切っても設定は保持されます。
- EEPROM消去（レイヤー2 + 左上端キー）を実行すると、すべての設定が工場出荷状態に戻ります。

## トラブルシューティング

### カーソルが動かない

1. USBケーブルを抜き差しして再起動してください。
2. トラックポイントのフラットケーブルが正しく接続されているか確認してください（[ビルドガイド](buildguide.md)参照）。

### カーソル速度が合わない

1. レイヤー2で速度プロファイルを切り替えて試してください（1～5キー）。
2. 固定プロファイルで好みに近いものを見つけてから、カスタムバンク（4または5）で微調整するのがおすすめです。

### カスタムバンクの調整ができない（赤3回点滅）

固定プロファイル（1～3）が選択されています。レイヤー2で「4」または「5」を押してカスタムバンクに切り替えてから調整してください。

### スクロールが効かない

1. レイヤー3が有効になっているか確認してください（LEDがプロファイル色で表示されていること）。
2. スクロールモードキーを押しながらトラックポイントを操作してください。

### 設定をリセットしたい

レイヤー2 + 左上端キー（Esc位置）でEEPROM消去を実行してください。すべての設定が工場出荷状態に戻ります。
