# OLSK60 ファームウェア

このフォルダにはOLSK60キーボードのファームウェアに関する案内が含まれています。

OLSK60 v2.1 は、デフォルトで **5-Split Space** レイアウト用のファームウェアが書き込まれた状態で出荷されます。他のレイアウトで使用する場合は、設定ツール（Remap / VIA / Vial）でキーマップを変更してください。

---

## どのファームウェア・ツールを使うか

ご利用状況に応じて、以下からお選びください。

### 1. 現在の状態のまま使う（更新しない）

**2026年6月までにご購入いただいた個体には、旧版のファームウェアが入っています。**
そのままご利用になる場合は、当面の間 Remap 用の設定を維持しますので、**Remap** をそのままご利用ください。ファームウェアの更新は不要です。

### 2. 最新版に更新して使う

最新版に更新する場合は、**VIA** または **Vial** のいずれかをお選びいただき、対応するファームウェアを書き込みます。

| 使いたいツール | 書き込むファームウェア | 補足 |
|----------------|----------------------|------|
| VIA | `techmechkeys_olsk60_via.uf2` | VIAに `olsk60_via.json` を読み込ませて使用します |
| Vial / Pipette | `techmechkeys_olsk60_vial.uf2` | キーボード定義はファームウェアに内蔵されています |

> **Vial をお使いの方へ:** 現在 Vial 公式アプリではロータリーエンコーダ周りの表示に不具合が確認されています。**Pipette では問題なく表示されるため、Pipette のご利用を推奨します。**

## 設定ツールについて

| ツール | 概要 | リンク |
|--------|------|--------|
| **Remap** | VIA互換の、Webブラウザ上で動作するキーマップ編集ツール。ソフトウェアのインストールが不要です。 | <https://remap-keys.app/> |
| **VIA** | キーマップ編集ツール。対応するキーボード定義ファイル（`olsk60_via.json`）を読み込んで使用します。 | <https://usevia.app/> |
| **Vial** | VIA系から派生したオープンソースの設定ツール。キーボード定義をファームウェア本体に内蔵するため、定義ファイルの読み込みが不要です。タップダンスやマクロなどにも対応します。 | <https://get.vial.today/> |
| **Pipette** | Vial互換キーボード向けの設定アプリ（Windows / macOS / Linux 対応）。USB経由でVial対応キーボードと通信し、キーマップやマクロ、ライティングなどを設定できます。本機ではVialのエンコーダ表示の不具合を回避できるため推奨します。 | <https://github.com/darakuneko/pipette-desktop> |

## 配布

ファームウェアバイナリ（`.uf2`）およびVIA用キーボード定義ファイル（`.json`）は、[GitHub Releases](https://github.com/techmech-keeb/OLSK60_v2/releases)から入手できます。

今回のリリースに含まれるファイル:

- `olsk60_via.json` — VIA用キーボード定義ファイル
- `techmechkeys_olsk60_via.uf2` — VIA用ファームウェア
- `techmechkeys_olsk60_vial.uf2` — Vial用ファームウェア

---

## 機能

このファームウェアは以下の機能を実装しています。

1. **設定ツール対応**
   - Remap / VIA / Vial（Pipette）に対応しており、キーマップをGUIでカスタマイズできます。

2. **標準的なキーボード機能**
   - 60%キーボードレイアウト対応（デフォルトは5-Split Space）
   - マルチレイヤーサポート
   - キーリマップ機能

3. **トラックポイント機能**
   - オートマウスレイヤー機能：トラックポイントを操作している間は自動的にキーマップのレイヤーが切り替わり、最下段のキーがマウス操作用に変わります。
   - トラックポイントの感度調整が可能です。
   - Depress機能：トラックポイントを軽く押し下げることでクリック操作が可能です。起動時に自動的に有効化されます。

## ファームウェア更新方法

> **補足:** 購入時点でファームウェアは導入済みのため、更新しない場合はこの操作は不要です（[「1. 現在の状態のまま使う」](#1-現在の状態のまま使う更新しない)を参照）。

1. キーボードをPCに接続していない状態にします。
2. ESCキーを押しながら、USBケーブルでキーボードをPCに接続します。
3. "RPI-RP2"という名前のドライブ（ボリューム）がPC上に表示されます。
4. [GitHub Releases](https://github.com/techmech-keeb/OLSK60_v2/releases)からダウンロードした `.uf2` ファイル（VIA用またはVial用）を"RPI-RP2"ドライブのルートディレクトリにコピーします。
5. コピーが完了すると、キーボードが自動的に再起動し、新しいファームウェアが適用されます。
6. VIA用ファームウェアを書き込んだ場合は、VIAに `olsk60_via.json` を読み込ませてご利用ください。Vial用ファームウェアの場合は、Vial（Pipette推奨）で接続してください。

---

## 注意事項

- ファームウェアの更新は自己責任で行ってください。
- 更新中はキーボードのケーブルを絶対に抜かないでください。
- 更新が終わり"RPI-RP2"ドライブの表示が消えたら更新完了です。
- 一度ケーブルを抜き差しして再起動してください。
